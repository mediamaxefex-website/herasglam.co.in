## 📖 Project Overview
This project is a high-end, responsive business website developed for **Hera's Glam**, a premium ladies beauty salon and makeup studio located in **Kurissumoodu, Changanacherry**. The design focuses on elegance, professional artistry, and local search visibility.

## 🛠️ Technical Stack
* **Frontend:** Semantic HTML5, Custom CSS3 Grid & Flexbox Layouts.
* **Typography:** Premium Google Fonts (Cormorant Garamond & Manrope).
* **Icons:** Font Awesome 6.5.1 Library.
* **SEO Engine:** JSON-LD LocalBusiness & BeautySalon Schema integration.

## 🚀 Key Features & SEO Strategy
* **Local SEO Dominance:** Integrated Geo-tags and JSON-LD Schema to ensure top ranking in Changanacherry local searches.
* **Direct Booking Integration:** One-click WhatsApp enquiry and calling features for instant lead generation.
* **Optimized Performance:** Clean, lightweight code without heavy frameworks for fast loading on mobile networks.
* **Visual Gallery:** Professionally curated showcase of bridal and treatment works.

## 📍 Business Information (Micro-data Included)
* **Location:** Thottasseril Building, Kurissumoodu, Changanacherry, Kerala.
* **Specialization:** Bridal Makeup, Hydra Facials, Hair Smoothening, and Cosmetology-focused care.
* **Operating Hours:** 09:30 AM - 08:00 PM (Monday - Sunday).
